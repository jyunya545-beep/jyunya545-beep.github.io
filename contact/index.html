お問い合わせ

サイト内容、ツールの不具合、権利関係、広告掲載などの連絡は、以下の内容を添えてご連絡ください。

連絡先

現在はGitHub経由での連絡を受け付けています。

GitHub Issuesで連絡する

連絡時にあると助かる情報

  • 対象ページのURL
  • 不具合や要望の内容
  • 利用している端末、ブラウザ、発生日時
  • 権利関係の連絡の場合は、対象コンテンツと確認できる情報

返信について

内容を確認のうえ、必要に応じて対応します。すべての連絡に返信できるとは限りませんが、サイト改善に関わる重要な内容は優先して確認します。

注意事項

個人情報、機密情報、公開してはいけない業務データは送信しないでください。ツールの利用に関する相談は、再現手順や入力例があると確認しやすくなります。
